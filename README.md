## PDF Texto + Stamp (Streamlit)

App Streamlit para:
- carregar um PDF
- desenhar uma área para TEXTO e/ou STAMP (por página)
- ver preview
- gerar um PDF final com texto + stamp

### Correr localmente
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

### Deploy no Streamlit Community Cloud
Repo: `fboliveira89-code/PDF_TEXT`
