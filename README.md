## PDF Texto + Stamp (Streamlit) — versão por cliques

Esta versão evita o problema do canvas ficar em branco no Streamlit Cloud:
- seleccionas a área com 2 cliques na imagem (canto sup. esq. + canto inf. dir.)
- funciona para TEXTO e STAMP
- stamp é convertido para PNG em memória antes de inserir

### Run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
